# Earthback — Explore Circle Categories

All prompts: no visible faces, documentary photograph, natural light, shallow depth of field.
Earthback palette: warm earth tones, terracotta, ochre, sage green, slate blue, undyed linen.

Run with:  `python queue-circles.py --category "earth-building"`
Run all:   `python queue-circles.py`
